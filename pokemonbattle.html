Pokemon Battle-A-Thon

This project uses JavaScript, HTML, SCSS, and Shell.

This project is a one page Pokemon game.

A player chooses a Pokemon character and competes against the character the computer chooses, which is chosen at random. The players battle it out choosing attack moves until one player wins by defeating the other at which time a new game can be played. Points are also tallied for the players.
