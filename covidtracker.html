Covid19 Tracker

This project uses Python, Powershell, JavaScript, CSS, HTML, and Batchfile.

This project was created using React to display the daily data of Covid19. The numbers displayed are based on the human data entered, and therefore is only as reliable as such.

This is a Tracker for the pandemic. It fetches the global data as well as the data based on country. The numbers fetched are the infected cases, the recovered cases and the cases of death reported. A line graph is displayed when retreiving global data, and a bar graph displays the numbers for each country based on the country picked by the user.
