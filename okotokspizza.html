Okotoks Pizza

This project uses HTML and SCSS.

This project is a five page website created for a ficticious pizza place out of Okotoks, AB.

It has a video that shows the fresh ingredients used in making pizza on the Home page. An interactive Googlemap for customer directions along with address and contact information on another page. An About page explaining the restaurant's background, and a menu page with specials the restaurant may have.
