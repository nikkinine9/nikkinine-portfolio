What I Do

* Web Page Design & Development *

My focus is on designing user-friendly websites that make people want to interact. The saying "always make a good first impression" could never be more applicable than when a potential client visits your website for the first time.

Web Development can be divided into 3 parts:

1. Front-End (Client-side scripting) - code that executes in a browser and determines what customers or clients will see when they land on a website.

2. Back-End (Server-side scripting) - code that executes on a web server and powers the behind-the-scenes mechanics of how a website works.

3. Database Technology - helps to keep data sorted by storing it in a centalized location and the website running smoothly and efficiently.


* Web Page Maintenance *

Keeping a website well maintained and attractive is important in order to engage customers. And keeping it up to date and relevent is a way to retain repeat business.


* Networking *

File sharing - you can easily share data between different users, or access it remotely if you keep it on other connected devices.

Resource sharing - using network-connected peripheral devices like printers, scanners and copiers, or sharing software between multiple users, saves money.
